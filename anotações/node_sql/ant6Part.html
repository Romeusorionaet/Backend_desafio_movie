Migrations

Vamos usar o migrations para automatizar a criação de tabelas

Na pasta migrations criamos um arquivo createUsers.js onde vai ser criado uma tabela de usuário. Será a parte responsável por criar o conteúdo mas nesse caso criamos a tabela users manual e que gere automaticamente mas não será gerado se esta já existir

Bcrypt

Temos que installar a bíblioteca Bcrypt para poder criptografar nossas senhas para que não fique visível no banco de dados. npm install bcrypt --save
Para gerar está criptografia temos que criar uma const {hash} = require("bcrypt") de dentro do bcrypt. E isso no UsersController.

Precisamos ainda no arquivo atribuir a uma const a função hash na qual importamos da bíblioteca.
Dentro dessa função preciamos passar o password e logo após o fator de complexidade de valor 8 por exemplo.
