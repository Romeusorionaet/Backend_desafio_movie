Esses dados devem está organizados por tabelas. Um banco de dados ele composto de tabelas e ela existe no banco de dados para organizar os dados

O modelo do banco de dados que vamos usar é o modelo de banco de dados relacional que usa tabelas que é composto por linhas e colunas.

Próximo passo é fazer a instalação do banco de dado sqlite que vamos usar. npm install sqlite3 sqlite --save Criei uma pasta database => sqlite => index.js aqui importamos o sqlite3 e sqlite do express e como vamos criar pela primeira vez usamos o async para criar o arquivo database automaticamente unsado o async antes da função queremos dizer que a função é acicrona.

Arquivo index.js dentro da pasta sqlite

const sqlite3 = require("sqite3"); 1 const sqlite = require("sqite"); 2 const path = require("path"); 3 async function sqliteConnection() { 4 const database = await sqlite.open({ 5 filename: path.resolve(__dirname, "..", "database.db"), 6 driver: sqlite3.database 7 }) return database 8 }

Lendo o arquivo a cima. 1 = estou importando o sqlite3 que contem os drives que vamos usar para estabelecer a comunicação 2 - estou importando o sqlite que vai ser responsável por conectar 3 - o node disponibiliza esse path que não precisamos instalar, já vem no node. vamos usar ele para indicar o caminho da pasta que vai ser instalado o arquivo database.db 4 - criamos uma função acicrona 5 - criamos uma const database que recebe uma promisse await, que vai suar o sqlite.open função essa que cria o database.db mas na linha abaixo irá expecificar onde será instalado 6 - esta especificando onde será instalado o arquivo indicando com ".." informando que se deve sair da pasta sqlite e em seguida instalando o database.db que ficará dentro da pasta database 7 - indica qual driver vai ser utilizado que é o sqlite3 8 - em fim esta retornando o const database

Agora vamos fazer a conexão com o banco de dados que será no server.js Arquivo criado: src => database => database.db
